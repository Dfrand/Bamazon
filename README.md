# Bamazon
CLI app that utilizes MySQL and Node.js
![Bamazon Customer Portal](Images/customerPortal.png)

## Getting Started
***

 - Clone repo.
 - Run 'npm install' in command line.
 - Run 'node index.js' to get started playing the game.


## Technologies Used:
***

 * Javascript
 * nodeJS
 * MySQL

##  Installs
***
 
### Inquirer
 - `npm install inquirer`
 
### MySQL
 - `npm install mysql`
